# Erickshaw Command Center - AI-native Operations Dashboard

This is an AI-native, mobile-first PWA dashboard designed for the Business Head of Yuma (EV battery swapping company).

## Features
- **Accountability Tracker**: Detects silence and missed deadlines automatically.
- **AI-Powered Chat**: Deep research into operational anomalies by cross-referencing multi-system logs.
- **Contextual Actions**: Drafts and sends WhatsApp follow-ups with one tap.
- **City Pulse**: Real-time KPI monitoring across 5 cities.

## Tech Stack
- Vanilla HTML/JS/CSS
- Gemini 2.0 Flash for AI logic
- PWA (Progressive Web App) architecture

## File Structure
- `abhilash-app/`: Core modular application (HTML/CSS/JS).
- `abhilash_yuma_pwa_demo.html`: Single-file PWA demo.
- `Abhilash.md`: Deep context on business operations.
- `rules.md`: Operational rules for the AI agent.
